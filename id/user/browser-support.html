Masalah Login

Jika Anda mengalami kesulitan log in ke account Anda, mohon pastikan bahwa cookie telah diaktifkan pada browser Anda. Website ini menggunakan cookies untuk menyimpan ID login dan password Anda (dalam format sandi) pada PC Anda selama sesi login.

Untuk mengaktifkan cookie, ikuti petunjuk di bawah ini untuk versi browser yang anda gunakan.

Binary (C.R.) S.A.

Chrome 40+

  • Kunjungi "Menu dropdown Chrome (Customize dan control Google Chrome)" dan pilih Settings.
  • Klik pada Show advanced settings.
  • Klik pada tombol Content settings dibawah menu Privacy.
  • Pilih Allow local data to be set (direkomendasikan).
  • Klik pada tombol Done untuk menyimpan settingan Anda.
Binary (IOM) Ltd

Mozilla Firefox 40+

  • Buka menu dan pilih Preferences.
  • Klik pada tab Privacy.
  • Set "Firefox will" ke "Remember history".
  • Klik pada tombol OK untuk menyimpan setingan Anda.
Binary (Europe) Ltd

Safari 5.x+

  • Kunjungi menu Safari dan pilih Preferences.
  • Klik pada tab Privacy.
  • Set "Blokir cookies:" ke Never.
  • Klik pada tombol X kanan atas untuk menyimpan dan menutup.
Binary (C.R.) S.A.

Opera 30+

  • Kunjungi menu Tools dan pilih Preferences.
  • Klik pada tab Advanced.
  • Pilih Cookies dari kolom sebelah kiri.
  • Pilih Accept cookies.
  • Klik pada tombol OK untuk menyimpan dan menutup.
Binary Ltd

Microsoft Internet Explorer 11+

  • Kunjungi menu Tools dan pilih Internet Options.
  • Klik pada tab Privacy.
  • Klik pada tombol Advanced.
  • Pilih Override automatic cookie handling.
  • Pilih Accept untuk keduanya First-party Cookies dan Third-party Cookies.
  • Klik pada tombol OK untuk menyimpan dan menutup.
Binary Ltd

Microsoft Edge 12+

  • Kunjungi menu More actions dan pilih Settings.
  • Dibawah Advanced Settings, klik pada tombol View advanced settings.
  • Dibawah bagian Cookies, pilih "Jangan blokir cookies".
  • Klik dimana saja diluar menu untuk menyimpan dan menutup.
